**Interactive Navbar with Rotation and Opacity Animation**

#This project demonstrates how to create an interactive navigation bar that rotates and changes opacity based on button clicks. It utilizes HTML, CSS, and JavaScript, with the GreenSock Animation Platform (GSAP) for smooth animations.

![Navbar Preview](https://github.com/goyalvansh4/InteractiveNavbar/blob/main/Screenshot%202024-01-19%20165603.png)


[![Live Preview](https://img.shields.io/badge/Live%20Preview-Click%20Here-blue)](https://goyalvansh4.github.io/InteractiveNavbar/)

#Features
Rotation Animation: Clicking on the buttons in the navigation panel triggers a rotation animation of the navigation bar.

Opacity Transition: The opacity of the navigation bar smoothly transitions when the buttons are clicked.

#Technologies Used
HTML
CSS
JavaScript
GSAP (GreenSock Animation Platform)

Usage
Click on the navigation buttons to see the rotation animation and opacity transition.
Explore the code to understand how the animations are implemented using HTML, CSS, and GSAP.
